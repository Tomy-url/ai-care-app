AI 요양원 운용 관리 시스템
GitHub Pages 배포용 최종 패키지

구성 파일
- index.html
- assets/style.css
- assets/app.js
- README.md
- README.txt

업로드 방법
1. GitHub 저장소 생성
2. 이 압축파일을 해제
3. 내부 파일과 assets 폴더를 저장소 최상위에 그대로 업로드
4. Settings > Pages > Deploy from a branch > main / root 선택

주의
- index.html만 올리면 안 됩니다.
- assets 폴더까지 같이 업로드해야 정상 동작합니다.
- 자동 보관은 같은 PC와 같은 브라우저 기준입니다.
