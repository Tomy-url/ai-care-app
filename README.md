# AI 요양원 운영관리 시스템

GitHub Pages 배포용 최종 패키지입니다.

## 폴더 구조

```text
index.html
assets/
  style.css
  app.js
README.md
README.txt
```

## GitHub Pages 업로드 방법

1. GitHub에서 새 저장소를 생성합니다.
2. 이 압축파일을 풀고, 안의 파일과 폴더를 저장소 최상위에 그대로 업로드합니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Source`를 `Deploy from a branch`로 선택합니다.
5. `Branch`는 `main`, 폴더는 `/ (root)`를 선택합니다.
6. 저장 후 몇 분 뒤 배포 주소가 생성됩니다.

## 중요

- `index.html`은 저장소 최상위에 있어야 합니다.
- `assets` 폴더를 반드시 함께 업로드해야 합니다.
- `index.html`만 단독 업로드하면 디자인과 기능이 정상 작동하지 않습니다.
- 자동 보관 기능은 같은 PC와 같은 브라우저 기준으로 동작합니다.

## 권장 운영 방식

- 평소에는 시스템 내부 자동 보관 기능 사용
- 주기적으로 `전체 기록 백업 저장` 실행
- 필요 시 엑셀용 저장 파일 별도 보관


## 문구 수정 사항

- 시스템 명칭을 `AI 요양원 운영관리 시스템`으로 통일했습니다.
